# XOAI Wheels

A collection of pre-built wheels for Python 3.14.

This package includes dependencies for the XOAI project.

## Included Packages

- pyyaml
- scipy
- pillow

## Installation

```bash
pip install xoai_wheels
```

## Usage

This is a meta-package that installs dependencies for XOAI projects. After installation, you can use:

```python
import yaml
import scipy
import PIL
```
